Solution:-

In the challenge, Track the Bed, We themed an idea of connecting Here Map API with Database of Local Hospitals along with a Beds Management System for the Hospital, where it can release/withhold beds, so that the end user can view beds are available or not. If available, they can register with their details for acquiring the bed. The Map will list all hospitals near the user from the database. There will be a registration option for listing new hospitals.
